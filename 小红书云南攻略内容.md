# 云南7天游超全攻略

## 标题：云南7天游保姆级攻略

## 正文内容：
📍云南7天深度游路线分享！
昆明→大理→丽江→香格里拉，一次玩转云南精华！

📅行程安排：
Day1：昆明
- 滇池看海鸥
- 云南民族村体验多民族文化
- 吃过桥米线

Day2：大理
- 大理古城漫步
- 洱海生态廊道骑行
- 品尝白族三道茶

Day3：大理深度
- 苍山索道俯瞰洱海
- 崇圣寺三塔拍照
- 喜洲古镇做扎染

Day4：丽江
- 丽江古城逛吃
- 狮子山观全景
- 腊排骨火锅必尝

Day5：玉龙雪山
- 冰川公园登顶
- 蓝月谷拍大片
- 《印象丽江》演出

Day6：香格里拉
- 普达措国家公园
- 独克宗古城转经筒
- 牦牛肉火锅

Day7：返程
- 松赞林寺祈福
- 购买伴手礼

💰预算参考：
人均6000-8000元（含交通住宿餐饮门票）

📌实用贴士：
1. 高原反应要预防，备好氧气瓶
2. 昼夜温差大，带够衣服
3. 紫外线强，防晒不能少
4. 尊重少数民族风俗

🍜美食推荐：
- 昆明：过桥米线、汽锅鸡
- 大理：烤乳扇、酸辣鱼
- 丽江：腊排骨、鸡豆凉粉
- 香格里拉：牦牛肉火锅、酥油茶

📸拍照打卡点：
1. 滇池海埂大坝
2. 洱海日出日落
3. 崇圣寺三塔倒影
4. 丽江古城夜景
5. 玉龙雪山冰川
6. 普达措属都湖
7. 松赞林寺金顶

💡行程优化：
时间充裕可加泸沽湖2日游
喜欢徒步推荐虎跳峡
